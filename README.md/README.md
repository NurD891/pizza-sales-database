#  Pizza Sales Database Project

Это мой проект по созданию и оптимизации базы данных для сети пиццерий.  
Я полностью построил БД в MySQL: от структуры таблиц до загрузки данных, индексов и аналитических запросов.

Проект полностью работает и его можно легко повторить на любом компьютере.

##  Как запустить проект

### Создание базы

Открыть файл:sql/01_ddl_schema/01_create_schema.sql

и выполнить его в MySQL Workbench.


### Загрузка данных

Все файлы из папки:sql/02_dml_load/


запускаются по очереди.  
Если нужно загрузить 'order_details_20000.csv':

1. ПКМ по таблице 'order_details'
2. Table Data Import Wizard
3. Выбрать CSV
4. Импортировать

### Индексация

Запустить файл:sql/03_indexes/01_create_indexes.sql

После этого запросы начинают работать намного быстрее.  
Разница хорошо видна через EXPLAIN.

### Проверка производительности

Для сравнения до/после:sql/04_queries_demo/01_explain_no_index.sql
                       sql/04_queries_demo/02_explain_with_index.sql


До индексов — полный скан таблиц.  
После индексов — 'ref', 'range', 'eq_ref', используются ключи, меньше строк.


### Аналитика

Запросы из:sql/04_queries_demo/04_analytics_queries.sql



Показывают:

1-самые популярные пиццы  
2-средний чек по филиалам  
3-анализ доставки  
4-что можно использовать как бандлы
